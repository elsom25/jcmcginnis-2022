# jcmcginnis.com

Personal website and blog for Jesse McGinnis, built with [Bridgetown 2.0](https://www.bridgetownrb.com/) and [Tailwind CSS 4](https://tailwindcss.com/).

**Live site:** https://jcmcginnis.com

## Tech Stack

- **Static Site Generator:** [Bridgetown 2.0](https://www.bridgetownrb.com/)
- **CSS Framework:** [Tailwind CSS 4](https://tailwindcss.com/) with @tailwindcss/typography
- **Templating:** ERB with custom Ruby helpers
- **Fonts:** [Inter](https://rsms.me/inter/) variable font
- **Deployment:** GitHub Pages via GitHub Actions

## Quick Start

```bash
# Install dependencies
bin/setup

# Start development server with CSS watching (localhost:4000)
bin/dev

# Build for production
bin/build
```

## Project Structure

```
├── bridgetown.config.yml    # Site configuration
├── config/initializers.rb   # Plugin init & custom helpers
├── Procfile.dev             # Foreman config for dev (Bridgetown + Tailwind)
├── bin/                     # Scripts: setup, dev, build
├── frontend/
│   └── styles/index.css     # Tailwind CSS 4 source
├── src/
│   ├── _layouts/            # ERB templates (base → default → post)
│   ├── _partials/           # Reusable components (header, footer)
│   ├── _posts/              # Blog posts in Markdown
│   ├── _data/               # Structured data (menu, site_metadata)
│   ├── assets/              # Syntax highlighting CSS, images
│   └── category.erb         # Prototype page for category archives
└── output/                  # Build output (gitignored)
```

## Writing Blog Posts

Create a new post in `src/_posts/` with the format `YYYY-MM-DD-slug.md`:

```yaml
---
title: "Your Post Title"
permalink: /blog/your-slug/
description: "Optional subtitle"
image: /assets/img/optional-header.jpg
home: true  # Include in homepage rotation
categories:
  - Leadership  # or: Musings, Talks
---

Your content in Markdown...
```

## Categories

Categories are auto-generated using Bridgetown's prototype pages feature. Adding a new category to a post's frontmatter automatically creates a category archive page at `/category/:slug/`.

Current categories: **Leadership**, **Musings**, **Talks**

## Custom Helpers

Defined in `config/initializers.rb`:

- **`featured_post`** - Returns a featured post for the homepage (recent if within 4 weeks, otherwise rotates daily)
- **`posts_by_year`** - Groups posts by year for the blog index

## Development

The development workflow uses [foreman](https://github.com/ddollar/foreman) to run Bridgetown and Tailwind CSS in parallel:

```bash
bin/setup  # Install dependencies
bin/dev    # Start dev server + CSS watcher
bin/build  # Production build
```

**Note:** `src/assets/css/style.css` is generated by Tailwind and gitignored. It's built automatically during `bin/dev` (watch mode) and `bin/build` (production).
